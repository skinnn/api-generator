## E-Commerce template  

###### Template for creating a basic e-commerce store.

+ #### Frontend  
	+ **Technologies**:
		+ Vue.js

+ #### Backend  
	+ **Technologies**:
		+ Node
		+ Express
		+ MongoDB
	+ **REST API** - [Documentation](https://github.com/skinnn/e-commerce/README.md#API-Documentation)
	+ **Token based authentication** (JWT)
	+ **Roles** (root, admin, user)
	+ **Integrated payment systems** (Paypal, Stripe)
	+ **Stripe customers**
